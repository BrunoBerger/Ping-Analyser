# Ping-Analyser
Continous ping analysis

### Install:
- pip install pythonping

### Arguments:
| Flag | Default | Description                     |
|------|---------|---------------------------------|
| `t`    | 10      | Test Duration in second                  |
| `i`   | 0.2     | Interval between each ping in s |
| `a`    | '8.8.8.8' | IP-adress                       |

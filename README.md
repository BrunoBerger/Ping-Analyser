# Ping-Analyser
Continous ping analysis

### Install:
- pip install pythonping
- pip install matplotlib

### Usage:
`py pingTest.py -t 20 -a IP_ADRESS_HERE`

### Arguments:
| Flag | Default | Description                     |
|------|---------|---------------------------------|
| `t`    | 10      | Test Duration in second                  |
| `i`   | 0.1     | Interval between each ping in s |
| `a`    | '8.8.8.8' | IP-adress                       |
